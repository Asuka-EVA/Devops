### uniq 去重

uniq test.txt    #相邻的相同行去重。
uniq -c test.txt #相邻的相同行去重并且计数。

### sort 排序

```
[root@oldboy ~]# sort test.txt >u.txt  #按数字顺序排序。
[root@oldboy ~]# uniq -c u.txt #整个文件实现了去重并且计数。
```

sort参数：
-n  按数字排序
-t  指定分隔符
-k  指定列  
-r  倒序
