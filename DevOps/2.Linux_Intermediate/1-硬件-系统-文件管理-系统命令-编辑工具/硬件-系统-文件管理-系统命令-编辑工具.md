### 文件类型说明

| 文件类型字母 | 类型含义                                       |
| ------------ | ---------------------------------------------- |
| -            | 普通文件(文本，二进制，压缩，图片，日志等)     |
| d            | 目录文件                                       |
| b            | 设备文件(块设备)存储设备硬盘/dev/sda,/dev/sr0  |
| c            | 设备文件(字符设备)，终端/dev/tty1              |
| s            | 套接字文件，进程与进程间的一种通信方式(socket) |
| l            | 链接文件                                       |

> flie命令能更加精准的判断文件的类型
