> 切片的循环遍历和数组的循环遍历是一样的

### 1、基本遍历

```go
package main
import "fmt"
func main() {
var a = []string{"北京", "上海", "深圳"}
for i := 0; i < len(a); i++ {
fmt.Println(a[i])
}
}
/*
北京
上海
深圳
*/
```

### 2、k，v遍历

```go
package main
import "fmt"
func main() {
var a = []string{"北京", "上海", "深圳"}
for index, value := range a {
fmt.Println(index, value)
}
}
/*
0 北京
1 上海
2 深圳
*/
```
