### 1、值类型

> 长度固定的类型

```go
bool
int(32 or 64), int8, int16, int32, int64//占用的内存不一样，避免浪费内存
uint(32 or 64), uint8(byte), uint16, uint32, uint64
float32, float64
string
complex64, complex128
array // 固定长度的数组
```

### 2、引用类型：(指针类型)

> 长度不固定，可以扩充的类型

```go
slice // 序列数组(最常用)
map // 映射
chan // 管道
```
