### 1、定义数字类型

```go
package main
import "fmt"
func main() {
var a int8 = 4
var b int32 = 4
var c int64 = 4
d := 4
fmt.Printf("a: %T %v \n", a, a)
fmt.Printf("b: %T %v \n", b, b)
fmt.Printf("c: %T %v \n", c, c)
fmt.Printf("d: %T %v \n", d, d)
}
/*
a: int8 4
b: int32 4
c: int64 4
d: int 4
*/
```

### 2、reflect.TypeOf查看数据类型

```go
package main
import (
"fmt"
"reflect"
)
func main() {
c := 10
fmt.Println( reflect.TypeOf(c) ) // int
}
```
