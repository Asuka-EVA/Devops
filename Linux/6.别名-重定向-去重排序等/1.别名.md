### 如何设置别名

提高输入效率：

例如

```bash
alias net='cat /etc/sysconfig/network-scripts/ifcfg-eth0'
```

防止误删数据，提醒用户

```bash
[root@oldboy ~]# alias rm='echo "你好，请不要使用rm命令，用mv替代"'
```

### 如何能不使用别名？

1.命令全路径执行

```
[root@oldboy ~]# /bin/rm a
```

2.命令开头加反斜线

```
[root@oldboy ~]# \rm -f *
```

3.取消别名

```
[root@oldboy ~]# unalias net
```
